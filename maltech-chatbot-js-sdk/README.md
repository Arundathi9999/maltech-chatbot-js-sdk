# MALtech Chatbot JavaScript SDK

A reusable, framework-agnostic JavaScript SDK for integrating with the MALtech AI Chatbot backend.

This SDK wraps the backend APIs and allows easy integration into any JavaScript-based application such as Angular, React, Vue, or Node.js.

---

## Installation

Copy the SDK folder into your project or install it as a local package.

```bash
npm install ./maltech-chatbot-js-sdk
